# D-ck-und-Flux-zeitreise-Agentur
Du bist ein Agent in der Dück und flux zeitreise Agentur. Per Retro-Terminal reist du von der Steinzeit bis in die Zukunft, um die Zeitlinie zu stabilisieren. Du erfüllst Missionen, sammelst XP für Level-Ups und weichst im Minispiel "Flux-Kopplung" Hindernissen aus. Bei Ausfällen rettest du das Terminal durch Hacker-Rätsel
